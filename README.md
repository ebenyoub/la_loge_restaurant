# La Loge Bar & Food

Socle technique de la refonte du site de La Loge Bar & Food à Lyon.

## Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- ESLint

Le projet est volontairement au stade d'initialisation : aucune page métier, interface d'administration, intégration d'images ou design final n'est développé à ce stade.

## Démarrer le projet

```bash
npm run dev
```

Le site est alors disponible sur [http://localhost:3000](http://localhost:3000).
